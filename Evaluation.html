Refugee and Immigrant Integration Dashboard (R2i)

Usability Evaluation


Dear friends,

Thanks for participating in this usability evaluation. The purpose of this exercise is to test if the web interface design is intuitive enough for users to browse around with no prior training.

Please answer the following questions and to me.

Open R2i

Group 1

  • In 2015, which host country had the highest total number of asylum seekers and refugees?
  • In this highest host country, which were the top 3 asylum seeker origin countries?

Group 2

  • In 2015, which origin countries had the highest total number of asylum seekers and refugees?
  • From this highest origin country, which were the top 3 host countries they fled to?
  • Were you able to download the selected records as a table?

Group 3

  • In the demographics page, which country had the highest number of persons of concern in 0-4 age group in year 2015?
  • What were the percentages of men and women?

Group 4

Background: Europe 2020 education headline target is at least 40% of 30 to 34-year-olds should have completed tertiary education.

  • In 2016, please name ONE host country's total tertiary education % had already met the target? (Hint:COB Total).
  • In some host countries the foreign country born population has higher tertiary education attainment % than the reporting country born population.
    Please review the host country United Kingdom in 2016, is this TRUE?

Group 5

  • What do you like the most about this R2i Dashboard?
  • What do you like the least? What would you suggest to improve it?

Thank You!
